# Vagrant Ansible Development

## Development

  1. Install Vagrant
  ```
  https://www.vagrantup.com/downloads.html
  ```

  2. Install Ansible on OSX
  ```bash
  sudo easy_install pip
  sudo pip install ansible --upgrade
  ```
  or upgrage Ansible to latest version
  ```bash
  sudo pip install ansible --upgrade
  ```

  3. Up vagrant
  ```bash
  vagrant up
  ```
